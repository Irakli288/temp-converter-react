# Конвертер температур (°C ↔ °F)

Реализовано в рамках **Контрольной работы №4**  
по дисциплине **«Технологии индустриального программирования»**  
Институт ИПТИП, кафедра индустриального программирования

**Студент:** Кекелашвили Иракли  
**Вариант №6**

Приложение корректно конвертирует:

- `0°C ↔ 32°F`
- `100°C ↔ 212°F`
- `-40°C ↔ -40°F`

## Технологии

- React (функциональные компоненты)
- `useState` для управления состоянием
- Чистая логика конвертации без внешних библиотек
- Vite — сборка и запуск

## Запуск

Проект создан в [StackBlitz](https://stackblitz.com).  
Для локального запуска (при наличии Node.js):

```bash
npm install
npm run dev
```
